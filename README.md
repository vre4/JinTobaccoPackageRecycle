# 微信小程序

塑封膜回收管理系统，主要由商户人员、回收人员、管理人员使用，其中商户人员使用箱子配置功能以及状态管理功能，回收人员通过地图的mark位置实时查看箱子状态，管理人员可获取每个箱子使用状态以及对塑封膜进行查询统计功能。分为登录模块、账户绑定模块、地理位置管理模块、箱子状态实时变更模块、地图回显模块、塑封膜回收与统计共六个模块。


## 使用过程

###登录
打开微信扫描二维码，进入塑封膜回收管理系统微信小程序
      
第一次登录需要选择登录的角色，初始化登录之后，不再需要选择角色登录。如果需要更换微信账户绑定角色，需要超级管理员登录后台管理页面操作，如下图

####账户绑定

####扫码配置
用户根据配置向导，可以一步一步的把箱子上的二维码识别号录入到系统中，一共有四步，图示如下：
 
点击下一步，即可扫码二维码。
 
扫码的结果会回显到系统中，如下图：
 
再次点击下一步，系统会把扫码的二维码编号，和目前的地理位置信息进行绑定，如下图：
 
点击地址绑定箱子即可完成绑定操作。
####管理绑定箱子
回到商户人员主页面，点击管理箱子，可以更改塑封箱子的状态。如下图：
 
###回收人员
回收人员第一登录需要绑定自己的角色，之后扫码进入系统，直接进入地图页面，地图上罗列着晋城市所有的回收箱子网点，回收人员可以点击网点晋城导航，或选择对箱子的塑封膜进行回收。
 
####回收箱子塑封膜
 
点击某个已经装满塑封膜的箱子，地图上回弹出选项卡，点击回收进入回首页面，如下图：
 
点击所对应的超市，回收塑封膜的总重量，点击确认按钮，即可完成回收操作。
####回收点导航
点击某个网点后，在弹出的选项卡上选择导航，即可使用地图进行导航。如下图：
 
系统会根据手机的位置作为初始位置，目的地自动设置为刚才所选网点，利用手机安装的地图即可完成导航。
